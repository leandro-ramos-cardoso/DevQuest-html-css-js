Instalação do Git

-> Você pode fazer o download do Git no site oficial.


Configurando usuário e email no GIT


  1. git config --global user.name "nome_usuario"
  2. git config --global user.email "seuemail@gmail.com"
  3. git config --list (Serve para verificar se a configuracao foi feita corretamente)

Passos GIT

  1. Crie um repositório Git. Um repositório é uma pasta que contém todos os arquivos do seu projeto e é gerenciado pelo Git. Você pode criar um repositório Git em um diretório existente usando o seguinte comando: git init
  2. git remote add origin https://github.com/leandro-ramos-cardoso/teste-del.git
  3. git clone
  4. git branch -M main
  5. git status
  6. git add .
  7. git commit -m "mensagem"
  8. git push -u origin main
